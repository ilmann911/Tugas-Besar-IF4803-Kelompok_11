# Tugas-Besar-IF4803-Kelompok_11
Rancanglah multi-linked list yang memodelkan hakim, terdakwa, dan hubungan di antara mereka. Seorang hakim dapat menghakimi banyak terdakwa, dan satu terdakwa hanya dapat dihakimi oleh satu hakim. Program ini ditujukan untuk mengelola kegiatan disuatu persidangan serta mencatat performa dari setiap keputusan hakim.
